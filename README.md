# Photo Card Project Readme

This project was made for my girlfriend and feel free to make your own or change some features.

## Features

This is a basic photo booth like application where you take photos download photo cards, as well as download each individual picture. I think this would be fun and entertaining.

### Updates for the future

I made this as a project to practice HTML, CSS, and JavaScript. I will be adding more to this and plan on keeping this side project active until I think it's in a good place. I will add a feature that will let you pick the frames and add more frames in the future.
